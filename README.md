## 🏠 House Price Prediction using Machine Learning

## 🌐 Live Demo
👉 [https://your-app-link.streamlit.app](https://house-price-prediction-ml-app.streamlit.app/)

Experience the live House Price Prediction application:

This interactive web application allows users to:
- Enter house details such as area, bedrooms, and location  
- Get real-time predicted house prices  
- Visualize data through graphs and heatmaps  
- Understand model predictions using SHAP explainability  

The application is built using Streamlit and powered by an XGBoost machine learning model.


## Overview
This project predicts house prices using machine learning techniques. It uses advanced models like XGBoost and includes data analysis, visualization, and explainability.

---

## Features
- Predict house prices in real-time
- Interactive web application using Streamlit
- Data visualization (graphs & heatmap)
- Location-based prediction (latitude & longitude)
- SHAP explainability for model interpretation

---

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Matplotlib, Seaborn
- SHAP

---

## Model Details
- Model Used: XGBoost
- Training Accuracy: **97.27%**
- Train-Test Split: 80% training, 20% testing
